# マインドマップエディター ユーザーマニュアル

## 1. 概要

マインドマップエディターは、アイデアを視覚的に整理し、AIとの対話を通じて発展させるためのツールです。直感的な操作でアイデアをマップ化し、AIの支援を受けながら思考を深めることができます。

![マインドマップエディター画面](https://via.placeholder.com/800x450)

## 2. 基本操作

### 画面構成

- **左側**: マインドマップ表示エリア
- **右側**: 編集パネル（マップ編集、ノード詳細、AIツール）
- **下部**: AIチャットエリア

### モード切替

- **ノードモード**: ノードの移動や編集が可能（デフォルト）
- **ズームモード**: マップの拡大・縮小・移動が可能
  - 「ノードモード（有効）」ボタンをクリックで切り替え
  - ズームコントロールは右下に常時表示

### テーマ切替

- ヘッダー右上の月/太陽アイコンでダーク/ライトテーマを切り替え

## 3. マインドマップの作成

### 新規マップ作成

1. **トピックから生成**:
   - 「マップ編集」タブの「トピックを入力」欄にテーマを入力
   - 「生成」ボタンをクリック

2. **チャット履歴から生成**:
   - AIとの会話を行った後「チャット履歴からマップ生成」をクリック

3. **手動作成**:
   - 「クリア」ボタンで新規マップを作成
   - ルートノードから子ノードを追加

### ノードの操作

- **ノード追加**: ノード上の「＋」アイコンをクリック
- **ノード削除**: ノード上の「✕」アイコンをクリック（ルートノードは削除不可）
- **ノード移動**: ノードをドラッグ＆ドロップ（ノードモード時）
- **ノード編集**: ノードをダブルクリックしてタイトルを編集

### ノードの詳細編集

1. ノードをクリックして選択
2. 「ノード詳細」タブで以下を編集:
   - タイトル
   - ノードタイプ（通常、アイデア、タスク、質問、メモ）
   - メモ（詳細情報）
3. 「保存」ボタンをクリック

### コンテキストメニュー

ノード上で右クリックすると表示:
- 子ノード追加
- 編集
- 削除
- AIに質問
- AIで展開

## 4. AI機能の活用

### AIチャット

- 画面下部のチャット入力欄にメッセージを入力
- AIはマインドマップの内容を理解した上で応答

### ノードの展開

1. ノードを選択
2. 「AIツール」タブの「選択ノードを展開」ボタンをクリック
   - または、コンテキストメニューから「AIで展開」を選択
3. AIが自動的に関連する子ノードを提案

### 関連アイデア提案

1. ノードを選択
2. 「AIツール」タブの「関連アイデア提案」をクリック
3. AIがチャットで関連するアイデアを提案

### マップの洞察抽出

「AIツール」タブの「洞察抽出」をクリックすると、AIがマップ全体を分析し、重要な洞察やパターン、次のステップを提案

### サジェスト機能

「AIツール」タブには選択中のノードに関連するサジェストが表示されます。クリックするとチャット入力欄に追加されます。

## 5. マップの更新と保存

### チャットからマップを更新

1. AIとのチャットを通じてアイデアを発展させる
2. 「チャットからマップ更新」ボタンをクリック
3. 更新内容の確認モーダルが表示される
4. 「更新する」ボタンでマップに反映

### 保存とエクスポート

- **保存**: ヘッダーの「保存」ボタンでブラウザに保存（次回訪問時に自動読み込み）
- **エクスポート**: 「マップ編集」タブの「エクスポート」ボタンでJSONデータを表示
  - 表示されたJSONをコピーして保存
- **インポート**: エクスポートしたJSONをテキストエリアに貼り付け、「インポート」ボタンをクリック

## 6. キーボードショートカット

| ショートカット | 機能 |
|-------------|------|
| Tab | 選択ノードに子ノード追加 |
| Delete | 選択ノード削除 |
| Enter | ノード編集 |
| Esc | 選択解除 |
| Ctrl + Z | 元に戻す |
| Ctrl + Y | やり直す |
| Ctrl + S | 保存 |
| Ctrl + F | 検索 |
| / | チャットにフォーカス |

## 7. 画面表示のカスタマイズ

### フォーカスモード

- ヘッダー右上の「フォーカスモード」ボタンをクリック
- マインドマップ以外の要素が半透明になり、集中して作業できます

### 自動レイアウト

- 「マップ編集」タブの「自動レイアウト」ボタンで、ノードを整理された配置に再配置

## 8. トラブルシューティング

### 一般的な問題と解決方法

- **マップが保存されない**: ブラウザのローカルストレージが無効になっていないか確認
- **AIが応答しない**: インターネット接続を確認し、ページをリロード
- **ノードが移動できない**: 「ノードモード」になっているか確認
- **エクスポートしたJSONが表示されない**: 「エクスポート」ボタンを再度クリック

### データのバックアップ

- 定期的にマップをエクスポートし、JSONデータをテキストファイルとして保存することをお勧めします

## 9. 応用テクニック

### 効果的なマインドマップ作成のヒント

1. **中心から始める**: 主要なトピックをルートノードに設定
2. **階層を意識する**: 関連性の強いアイデアを近くに配置
3. **ノードタイプを活用**: 「アイデア」「タスク」「質問」など目的に応じたタイプを設定
4. **AIと対話しながら発展**: チャットでアイデアを深掘りし、マップに反映
5. **定期的に再構成**: 「マップ再構成」機能を使って論理的な構造を保つ
